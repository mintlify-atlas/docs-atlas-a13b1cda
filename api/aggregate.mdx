## Overview

The `Aggregate` class is the base class that provides read operations for querying aggregated data. It implements a sorted key-value store where you can efficiently query counts, sums, and ranges of items.

The data structure stores:
1. **Key**: Any Convex value (number, string, array, etc.)
2. **ID**: A unique string identifier
3. **SumValue**: An optional number that is aggregated by summing (defaults to 0)

## Class Signature

```typescript
class Aggregate<
  K extends Key,
  ID extends string,
  Namespace extends ConvexValue | undefined = undefined,
>
```

### Type Parameters

<ParamField path="K" type="Key">
  The type of keys used to sort and query items. Can be any Convex value type.
</ParamField>

<ParamField path="ID" type="string">
  The type of unique identifiers for items.
</ParamField>

<ParamField path="Namespace" type="ConvexValue | undefined" default="undefined">
  Optional namespace type for multi-tenant data isolation.
</ParamField>

## Constructor

```typescript
constructor(component: ComponentApi)
```

<ParamField path="component" type="ComponentApi" required>
  The Convex Aggregate component API instance.
</ParamField>

### Example

```typescript
import { components } from "convex/_generated/api";
import { Aggregate } from "@convex-dev/aggregate";

// Create an aggregate instance
const aggregate = new Aggregate<number, string>(components.aggregate);

// Query count of items
const count = await aggregate.count(ctx);
```
